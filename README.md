# @pylogmonmcp/powerpoint-generator

A PowerPoint generator for Model Context Protocol (MCP) using Bun (Note: Unable to read/edit existing presentation)

## Tools

- create-presentation
- add-slide
- add-text
- add-table
- add-shape
- add-chart
- get-file-url

## Start Server

```bash
bunx -y @pylogmonmcp/powerpoint-generator
```
